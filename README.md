# Word Forecast Max

A simple React/Vite app that forecasts valid next letters while building English words.

## Run it locally

1. Install Node.js from https://nodejs.org
2. Open Terminal or Command Prompt in this folder
3. Run:

```bash
npm install
npm run dev
```

4. Open the local URL Vite gives you, usually:

```text
http://localhost:5173
```

## Build for sharing

```bash
npm run build
```

You can deploy the folder to Vercel, Netlify, or another static web host.

## Notes

The current app includes a starter dictionary in `src/main.jsx`. Replace the `WORDS` array with a larger English word list for production use.
